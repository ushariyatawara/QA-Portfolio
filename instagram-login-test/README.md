# Instagram Login Feature - Manual Testing Documentation
This repository contains a set of manual artifacts created for **Instagram Login** feature to demonstrate QA practices including requirement analysis, test scenario creation, test case writing, test reporting and defect tracking.

## Project Overview
The purpose of this project is to validate the functionality of the Instagram Login with different input conditions. It includes verifying credentials, validation messages, password reset and third-party login.
